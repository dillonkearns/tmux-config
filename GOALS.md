# Active Project Goals

Track what you're working on across repositories. This keeps you focused when jumping between projects on the tablet. Goals are also tracked in `.claude-status.json` per-repo (via Claude Code hooks + global CLAUDE.md instructions).

## How to use

Each entry is a project with a concrete goal. When you finish a goal, move it to Done or replace it. Keep this list short — if everything is a priority, nothing is.
